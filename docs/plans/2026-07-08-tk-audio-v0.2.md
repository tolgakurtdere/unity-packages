### Task 1: One-shot SFX enhancements — voice cap, delayed play, StopSfx (one-shot side)

**Files:** `Runtime/AudioCatalog.cs` (add field), `Runtime/SfxPlayer.cs`, `Runtime/AudioService.cs` (overload + StopSfx/StopAllSfx), `Tests/Editor/AudioServiceTests.cs`.

- **Catalog:** add `[SerializeField] private int maxConcurrentVoices;` (`[Min(0)]`, tooltip "0 = unlimited; when a key at its cap plays again, the oldest voice of that key is culled") + `public int MaxConcurrentVoices => maxConcurrentVoices;`.
- **SfxPlayer:** `ActiveShot` gains a `string Key` (empty for direct clips). In the catalog `Play`, after the throttle passes: if `entry.MaxConcurrentVoices > 0`, count live shots with `Key == entry.Key`; while `count >= cap`, cull the oldest (stop + return its source, remove from `_active`). Then spawn. Add `PlayResolved(clip, baseScale, pitch, key)` (key threaded through; direct overload passes `null`/empty).
- **Delayed overload:** `AudioService.PlaySfx(string key, float volumeScale, float delaySeconds)` → `SfxPlayer.Play(entry, volumeScale, delaySeconds)`; when `delaySeconds > 0`, an `async void` unscaled wait (disposed-guarded) then the normal path. Throttle is stamped at call time (before the wait). Keep the existing 2-arg `PlaySfx(key, volumeScale=1)` delegating with delay 0.
- **StopSfx/StopAllSfx (one-shots now; loops in Task 2):** `AudioService.StopSfx(string key)` → stop+return every active shot whose `Key == key`; `StopAllSfx()` → all. Add `SfxPlayer.StopByKey(key)` / `StopAll()`.
- **Tests:** cap culls oldest at the limit (spawn cap+1 of a key → count stays cap); direct clips uncapped; `PlaySfx(key, 1, 0.2f)` spawns nothing on the same frame; `StopSfx(key)` stops only that key; `StopAllSfx` clears all one-shots.
- [ ] Implement + gate + commit `feat(audio): per-key voice cap, delayed PlaySfx, StopSfx/StopAllSfx (one-shots)`.

### Task 2: Loop SFX + `AudioHandle`

**Files:** `Runtime/AudioHandle.cs` (new), `Runtime/SfxPlayer.cs`, `Runtime/AudioService.cs`, `Runtime/Audio.cs` (façade mirrors), `Tests/Editor/AudioLoopTests.cs` (new).

- **`AudioHandle`** (public readonly struct): private fields `SfxPlayer _owner` (internal type in a private field — legal) + `int _id`. `IsPlaying => _owner != null && _owner.IsLoopPlaying(_id)`; `Stop() => _owner?.StopLoop(_id)`; `FadeOutAndStop(float seconds) => _owner?.FadeLoop(_id, seconds)`. `internal AudioHandle(SfxPlayer, int)`.
- **SfxPlayer loop registry:** `Dictionary<int, LoopVoice>` (`LoopVoice { AudioSource source, float baseScale, int fadeGen }`), `int _nextLoopId`, a dedicated loop-source list grown on demand (own `GameObject` children under the host, NOT the one-shot pool). `int StartLoop(AudioClip, baseScale, pitch, key)`: acquire a loop source, `loop=true`, volume via `EffectiveSfxVolume`, play, register, return id (0 = none when `EffectiveSfxVolume<=0` → still register so unmute can bring it? NO — v0.2 keeps it simple: don't start a loop while the channel is silent, return default handle; documented). `StopLoop(id)`, `IsLoopPlaying(id)`, `FadeLoop(id, seconds)` (generation-guarded async fade → stop+recycle). `ApplyVolumes` also sweeps loop sources. Loop `LoopVoice` carries `key` for StopByKey; extend `StopByKey`/`StopAll` to loops.
- **AudioService:** `AudioHandle PlaySfxLoop(string key)` (resolve entry, wrong-channel warn tolerated, → `_sfx.StartLoop(...)` wrapped in a handle) + `AudioHandle PlaySfxLoop(AudioClip clip, float volumeScale=1, float pitch=1)`.
- **Audio façade:** mirror both `PlaySfxLoop` overloads (return `AudioHandle`; unbound → `default`).
- **Tests:** `default(AudioHandle)` Stop/IsPlaying safe; `PlaySfxLoop` uses a dedicated loop source (host child count; distinct from the one-shot pool), `IsPlaying` true; `Stop()` recycles it, `IsPlaying` false; a second `Stop()` and a stale handle are no-ops; `PushMute` zeros a running loop's volume; `SfxEnabled=false` → `PlaySfxLoop` returns a non-playing handle.
- [ ] Implement + gate + commit `feat(audio): looping SFX with AudioHandle (Stop/FadeOutAndStop/IsPlaying)`.
